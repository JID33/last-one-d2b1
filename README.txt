# D2B1 Auth App

Ce projet contient un mini-système d'authentification basé sur `localStorage`.

## Fichiers

- `index.html` : Page d'inscription.
- `login.html` : Page de connexion.
- `admin.html` : Tableau de bord administrateur.
- `style.css` : Style commun.
- `script.js` : Logique JS.
- `README.txt` : Explication rapide.

Connexion à une base de données réelle (Firebase, MongoDB, MySQL) et hébergement sur GitHub ou Render.
